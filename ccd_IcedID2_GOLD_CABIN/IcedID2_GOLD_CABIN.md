## MemprocFS NTFS Timeline Flag Breakdown

The format for each entry is:

```
"timestamp", NTFS, [FLAG], [field], [field], [hex], [hex], "path"
```

Here are the flags between `NTFS` and the hex values:

**Record Type**

| Flag | Meaning |
| --- | --- |
| `NTFS` | Indicates this event comes from the NTFS file system journal/MFT |

**Operation Flags**

| Flag | Meaning |
| --- | --- |
| `CRE` | **Create** — the file/directory was created |
| `MOD` | **Modify** — the file data or metadata was modified |
| `RD` | **Read** — the file was read/accessed |

**Numeric Fields (after the operation flag)**

These three fields appear in order:

| Position | Example | Meaning |
| --- | --- | --- |
| 1st number | `0`, `0x42` | **Flags/Attributes** — NTFS file attribute flags (e.g. `0x42` = hidden + archive) |
| 2nd hex | `0x0`, `0x14f00` | **MFT Entry number** — the inode/record number in the Master File Table |
| 3rd hex | `0x12469a400`, `0x86c37800` | **File size** in bytes (hexadecimal) |

**Summary of a full example:**

```
"2024-06-15 08:56:20", NTFS, RD, 0, 0x14f00, 0x86c37800, "\1\Users\admin\Downloads\invoice_173.zip"
```

- **RD** → file was **read**
- **0** → no special NTFS attributes
- **0x14f00** → MFT record number
- **0x86c37800** → file size (~2.26 GB in this case)
- Then the **full file path**

Q5- Determining the root of the malicious activity is essential for comprehending the extent of the intrusion. What is the malicious command that triggered this malicious behavior?

**Answer**: `rundll32.exe dar.dll,DllRegisterServer`

**Explanation**: Volatility’s `pstree` output shows a suspicious invocation of `rundll32.exe`, a legitimate Windows binary commonly abused to load malicious DLLs. Here, `DllRegisterServer` mimics a legitimate DLL registration export, but the malicious logic runs inside the DLL’s `DllRegisterServer` implementation. PID 3312 corresponds to the last observed execution.

```powershell
*** 3312	4460	rundll32.exe	0xc60db933e080	5	-	1	False	2024-06-15 08:59:52.000000 UTC	N/A	\Device\HarddiskVolume3\Windows\System32\rundll32.exe	"C:\Windows\System32\rundll32.exe" dar.dll,DllRegisterServer	C:\Windows\System32\rundll32.exe
```
